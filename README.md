# PyMon

An education web application meant to teach Python programming while providing a dynamic learning experience with Artificial Intelligence.

### Technologies
- **Database:** Supabase Database

- **Authentication:** Supabase Auth

- **Hosting:** Netlify

- **Server:** Netlify

- **Large Language Model:** OpenAI gpt-4o-mini, OpenAI gpt-3.5-turbo, OpenAI text-embeddings-3-small

## To run, run these commands in the terminal:

1. `cd ivr-app`
2. `npx tailwindcss -i ./src/app.css -o ./src/app.css --watch`
3. `npm install netlify-cli -g`
4. `npm install`
5. New Terminal
6. `netlify dev`

If you are getting a blank page, make sure your supabaseURL is defined on `utilities/Supabase.js`
